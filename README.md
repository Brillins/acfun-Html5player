# acfun-Html5player
acfun-html5player是一个为AcFun设计的全新Html5第三方播放器，完全按照官方Flash播放器样式设计，完整支持Flash版本的所有快捷键。

暂时只支持简单的弹幕发送，不支持拖动小窗口播放，可以播放普通弹幕。

这个项目是一个开源项目，遵循MIT开源协议。要联系作者请发邮件至 xyc0714@aliyun.com 。
